### 第 1 步：配置 DeepSeek API Key ⚠️ 必须

**为什么**: AI 分类、摘要、打分都需要这个 Key

**怎么做**:
1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 创建 API Key
4. 编辑 `.env` 文件，填入：
   ```bash
   DEEPSEEK_API_KEY=sk-your-actual-api-key-here
   ```

**时间**: 2 分钟

### 第 3 步：配置 MCP Server（可选但强烈推荐）⭐

**为什么**: 让 Claude Desktop 能用自然语言查询情报

**怎么做**:

1. **安装 MCP 依赖**（已完成✓）
   ```bash
   cd mcp-server && npm install
   ```

2. **配置 Claude Desktop**
   
   找到配置文件：
   - **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
   - **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
   
   添加配置：
   ```json
   {
     "mcpServers": {
       "alpha-radar": {
         "command": "node",
         "args": ["server.js"],
         "cwd": "C:/Users/lenovo/alpha-radar/mcp-server",
         "env": {
           "API_BASE_URL": "http://localhost:3001/api",
           "API_SECRET": ""
         }
       }
     }
   }
   ```

3. **重启 Claude Desktop**

4. **开始对话**
   ```
   获取过去 24 小时最重要的香港合规新闻
   
   搜索所有提到 SFC 和牌照的新闻
   
   过去一周各分类的情报数量分布如何？
   ```

**时间**: 5 分钟

## 📋 可选配置（按需）

### 配置更多推送渠道

#### 飞书（国内团队推荐）
```bash
FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
FEISHU_SECRET=xxx
```

#### ntfy（个人用户，超简单）
```bash
NTFY_TOPIC=my-alpha-radar
NTFY_SERVER=https://ntfy.sh
```
无需注册，直接用浏览器访问 https://ntfy.sh/my-alpha-radar 查看推送

#### Bark（iOS 用户）
1. App Store 下载 Bark
2. 打开 App 获取 Device Key
3. 填入 `.env`:
   ```bash
   BARK_DEVICE_KEY=your-device-key
   ```
